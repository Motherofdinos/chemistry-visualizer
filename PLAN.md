# План покращення Chemistry Visualizer

## 1. Нові теми та реакції

### Поточні теми (вже є)
- Властивості кисню (прості речовини): H₂ + O₂, C + O₂
- Властивості кисню (складні речовини): CH₄ + O₂
- Хімічні властивості спиртів: C₂H₅OH + O₂

---

### Нові теми для додавання

#### Тема: Вода та її властивості
| Реакція | Рівняння |
|---|---|
| Розклад води | 2H₂O → 2H₂ + O₂ |
| Взаємодія натрію з водою | 2Na + 2H₂O → 2NaOH + H₂ |
| Взаємодія оксиду кальцію з водою | CaO + H₂O → Ca(OH)₂ |

#### Тема: Оксиди та кислоти
| Реакція | Рівняння |
|---|---|
| Утворення сульфатної кислоти | SO₃ + H₂O → H₂SO₄ |
| Утворення вугільної кислоти | CO₂ + H₂O → H₂CO₃ |
| Взаємодія NO₂ з водою | 3NO₂ + H₂O → 2HNO₃ + NO |

#### Тема: Кислоти та основи (нейтралізація)
| Реакція | Рівняння |
|---|---|
| HCl + NaOH | HCl + NaOH → NaCl + H₂O |
| H₂SO₄ + 2NaOH | H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O |
| HNO₃ + KOH | HNO₃ + KOH → KNO₃ + H₂O |

#### Тема: Вуглеводні (Алкани)
| Реакція | Рівняння |
|---|---|
| Горіння метану | CH₄ + 2O₂ → CO₂ + 2H₂O (вже є, покращити) |
| Горіння пропану | C₃H₈ + 5O₂ → 3CO₂ + 4H₂O |
| Горіння бутану | 2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O |

#### Тема: Синтез аміаку та азотні речовини
| Реакція | Рівняння |
|---|---|
| Синтез аміаку (Габер) | N₂ + 3H₂ → 2NH₃ |
| Окиснення аміаку | 4NH₃ + 3O₂ → 2N₂ + 6H₂O |
| Розклад аміаку | 2NH₃ → N₂ + 3H₂ |

#### Тема: Окисно-відновні реакції
| Реакція | Рівняння |
|---|---|
| Залізо + кисень | 3Fe + 2O₂ → Fe₃O₄ |
| Магній + кисень | 2Mg + O₂ → 2MgO |
| Алюміній + кисень | 4Al + 3O₂ → 2Al₂O₃ |

#### Тема: Хімічні властивості карбонатів
| Реакція | Рівняння |
|---|---|
| Розклад карбонату кальцію | CaCO₃ → CaO + CO₂ |
| Взаємодія з кислотою | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂ |

## 3. Теми оформлення (Color Themes)

**Проблема:** На чорному фоні не видно чорних атомів карбону (C).

### Запропоновані теми:

| Назва | Фон сцени | Фон сторінки | Колір C | Підходить для |
|---|---|---|---|---|
| **Dark** (поточна) | `#111111` | темно-сірий | `#444444` + обводка | Виглядає "науково" |
| **Light** | `#e8f4f8` | світло-блакитний | `#222222` | Максимальна видимість C |
| **Lab** | `#1a2a1a` | темно-зелений | `#888888` | Стиль лабораторії |
| **Space** | `#050510` | темно-синій | `#aaaaaa` (сіро-білий) | Ефектний вигляд |

### Реалізація:
- Кнопки-перемикачі теми у sidebar (іконки або radio buttons)
- CSS змінні (`--scene-bg`, `--sidebar-bg`, `--primary-color`) для швидкого перемикання
- Збереження вибраної теми в `localStorage`
- Колір атому C змінюється залежно від теми (темний на світлому, світлий на темному)

## 4. Анімація переходу реагенти → продукти

**Головна ідея:** Показати, як атоми з молекул реагентів "перебудовуються" у молекули продуктів. Кожен атом — це окремий об'єкт, який переміщується у 3D-просторі.

### Алгоритм анімації:

```
1. Показати реагенти (початкова позиція кожного атома)
2. Користувач натискає "Показати продукти"
3. Атоми "розлітаються" (фаза руйнування зв'язків) — 0.5с
4. Атоми переміщуються на нові позиції (фаза перебудови) — 1.0с
5. Нові зв'язки "з'являються" (фаза утворення продуктів) — 0.5с
```

### Технічна реалізація (Three.js + TWEEN або власний lerp):

```js
// Atom mapping: який атом реагента → який атом продукту
// Зберігається в reactions.js разом з реакцією
atomMapping: [
  { from: { molecule: 'H2', atom: 0 }, to: { molecule: 'H2O', atom: 1 } },
  { from: { molecule: 'H2', atom: 1 }, to: { molecule: 'H2O', atom: 2 } },
  // ...
]
```

### Що додати в `reactions.js`:

Кожна реакція отримає поле `atomMapping` — масив відповідностей атомів реагентів → продуктів. Це дозволить точно анімувати переміщення кожного атома.

### Фази анімації:

1. **Розрив зв'язків** — циліндри-зв'язки зникають (fade out)
2. **Переміщення атомів** — плавний рух (easing: easeInOutCubic)
3. **Утворення нових зв'язків** — циліндри з'являються (fade in)

### Підсвічування атомів за типом:
- Під час анімації атоми підсвічуються кольором свого типу (H — синій ореол, O — червоний, C — оранжевий)
- Після завершення анімації підсвічування зникає

## 5. Додаткові покращення UI

### Легенда атомів
- Внизу sidebar показати кольорову легенду: ⚪ H, 🔴 O, ⚫ C, 🟣 Na, 🟡 S, тощо
- Видимість легенди залежить від атомів поточної реакції

### Інформація про молекули
- При кліку на молекулу у 3D — показати попап з назвою, формулою, молярною масою
- Виділення молекули при наведенні (hover glow)

### Кнопка "Анімувати реакцію"
- Окрема кнопка для запуску анімації переходу
- Прогрес-бар або таймер анімації
- Кнопка "Повторити" після завершення

### Швидкість анімації
- Слайдер "Повільно / Нормально / Швидко"
